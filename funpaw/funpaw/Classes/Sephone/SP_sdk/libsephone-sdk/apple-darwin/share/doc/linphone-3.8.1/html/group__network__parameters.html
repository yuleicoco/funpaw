void sephone_core_set_network_reachable ( SephoneCore *  lc,
bool_t  value 
)

This method is called by the application to notify the sephone core library when network is reachable. Calling this method with true trigger sephone to initiate a registration process for all proxies. Calling this method disables the automatic network detection mode. It means you must call this method after each network state changes.
